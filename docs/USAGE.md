# 使い方

## 基本的なセットアップ

ボコーダーは **キャリア**（音色となる波形）と **モジュレーター**（スペクトル形状を提供する声など）の2入力で動作します。

### DAW での使用

1. インストゥルメントトラックにプラグインを挿入
2. サイドチェイン入力（Input 3/4）にマイクや録音済みボーカルのトラックを接続
3. MIDI キーボードを弾くとボコーダーが鳴る

> DAW によってサイドチェインの設定方法が異なります。Logic Pro では "Side Chain" ドロップダウン、Ableton Live では "Audio From" で設定します。

### スタンドアロンでの使用

1. Drawwave Vocoder.app を起動
2. Options → Audio/MIDI Settings でオーディオデバイスを選択
3. 接続したマイクが Channel 1/2 としてモジュレーター入力になる
4. 画面下部のバーチャルキーボードまたは MIDI キーボードで演奏

**コンピューターキーボードでも演奏できます（スタンドアロン）**

```
z s x d c   v g b h n j m    ← C3〜B3
q 2 w 3 e   r 5 t 6 y 7 u    ← C4〜B4
i 9 o 0 p                    ← C5〜E5
```

## 描画エディター（左列）

3つの描画エリアが縦に並んでいます。共通操作：

| 操作 | 内容 |
|---|---|
| 左ドラッグ | 描く |
| 右クリック | メニュー（Reset / Smooth / Invert ほか） |

### キャリア波形エディター（上段）

MIDI で鳴るシンセサイザーの波形を描くエリア。描いた波形はリアルタイムで音に反映されます。

プリセットボタン: **Sine / Saw / Square / Tri / Pulse**

### バンドゲインエディター（中段）

16バンドのスペクトル整形を棒グラフで編集するエリア。

- `1.0` がフラット（±0dB）、`2.0` で +6dB ブースト、`0.0` で完全カット
- バーに薄く重なる表示がリアルタイムの各バンドレベル（VUオーバーレイ）

プリセットボタン: **A / I / U / E / O**（Bark スケールベースの母音フォルマント近似）、**Flat**（全バンド均一）

### LFO シェイパーエディター（下段）

変調波形を描くエリア。再生中は縦線インジケーターが LFO の現在位置を示します。エリア右上に現在の Rate・Depth・Dest が表示されます。

プリセットボタン: **Sine / Tri / Ramp / Rand**

## ノブパネル（右上）

### LFO セクション

| コントロール | 説明 |
|---|---|
| Rate | LFO の速度（0.01〜20 Hz） |
| Depth | 変調の深さ |
| Dest | 変調先：Formant（フォルマントシフト）/ Tilt（バンドゲイン全体の傾き）/ Pitch（キャリアピッチ） |
| Retrig | MIDI ノートオンで LFO をリセットするか |
| Sync | テンポ同期 ON/OFF（DAW プラグインのみ） |
| Div | テンポ同期時の音符割り（4/1〜1/32）（DAW プラグインのみ） |

### Vocoder セクション

| コントロール | 説明 |
|---|---|
| Mic Gain | モジュレーターの入力ゲイン |
| Frmt | フォルマントシフト（±12 semitones）。プラスで高く細い声、マイナスで低く太い声 |
| Atk | エンベロープフォロワーのアタック。短いと子音がくっきり |
| Rel | エンベロープフォロワーのリリース。長いほどなめらかにぼける |
| Bands | バンド数（8 / 16 / 24 / 32） |

### Output セクション

出力は `ボコーダー出力 + Mic Mix × 原音マイク + Car Mix × シンセ（声量連動）` の加算ミックスです。

| コントロール | 説明 |
|---|---|
| Mic Mix | 原音マイクのミックス量。ボコーダー出力に重ねてマイク生声を足す（0 = 足さない） |
| Car Mix | シンセのミックス量。声のエンベロープに連動してシンセ原音を足す（0 = 足さない） |
| HP | ハイパスフィルター（20〜500 Hz）。低音のこもりを取る |
| LP | ローパスフィルター（2k〜20k Hz）。高域を丸める |
| Gain | マスターゲイン（-60〜+12 dB） |
| Noise | ノイズミックス。ホワイトノイズを混ぜて子音（s, t, sh）を強調する |

ノブパネルの下にある **MIC IN / CARRIER / OUT** のレベルメーターで信号の流れを確認できます。

## プリセットブラウザー（右下）

リストからプリセット名をダブルクリックするとロードされます。現在のプリセットは名前が太字＋シアンのラインで示されます。

| ボタン | 説明 |
|---|---|
| Save | 現在の状態に名前をつけて `.dvoc` ファイルに保存 |
| Delete | 選択中のユーザープリセットを削除（ファクトリーは削除不可） |
| Cap A / Cap B | 現在の状態をスロット A または B にキャプチャ |
| Rec A / Rec B | スロット A または B の状態を呼び出す |

Cap/Rec の A・B スロットを使うと、2つの音色をすばやく行き来して比較・切替ができます。

## 音作りのヒント

**声がうまく聞こえない**
- Noise を 10〜30% に上げると子音（s, t, sh）が明瞭になります
- Rel を短くする（30〜80ms）と言葉がはっきりします
- Frmt を ±1〜3 semitones 動かすと声と馴染みやすくなります

**音が薄い・迫力がない**
- バンドゲインの低域〜中域（左側バー）を持ち上げます
- Bands を 24 か 32 に増やすと解像度が上がり密度感が出ます
- Frmt を - 方向にずらして重心を下げてみてください

**ロボット感を強くしたい**
- Bands を 24 以上に増やす
- Rel を短く（20〜40ms）、Atk を速く（1〜5ms）
- Frmt を +2〜4 に設定

## ファクトリープリセット

| プリセット | キャリア | 特徴 |
|---|---|---|
| **Init** | Saw | フラットなデフォルト状態。音作りのスタート地点 |
| **Classic Robot** | Pulse | Q 高め・高速 Env・Frmt +2。細くナザールな Pulse でロボット感を強調 |
| **Daft Style** | Saw | LFO（*カスタム：ソフトスクエア*）が Formant を揺らす（0.5Hz / Depth 30%） |
| **Whisper Chorus** | Sine | Noise 70%。純音の下地に子音が立つブレシーなコーラス感 |
| **Talking Pad** | Sine | LFO（*カスタム：ブレス型*）が Formant を有機的に揺らす（0.2Hz）。長い尾引きパッド |
| **Buzz Lead** | Square | 24バンド・Q=9・超速 Env（1ms/20ms）。ギラギラしたリード |
| **Vintage 70s** | *カスタム（ソフトクリップ Saw）* | 8バンド・Q=3。tanh で丸めた Saw、アナログ的な温かみ |
| **Choir** | *カスタム（sine + 2・3倍音）* | LFO（*カスタム：非対称ビブラート*）が Band Tilt を揺らす（0.8Hz）。オルガン的なコーラス感 |
| **Spectral Wash** | Triangle | LFO（*カスタム：極スロー sine*）が Band Tilt を微動（0.15Hz / Depth 25%）。Triangle の滑らかさでにじむアンビエント質感 |
| **Glitch** | Pulse | LFO（*カスタム：8段ステップ波*）が Formant を高速変調（8Hz / Retrig ON）。Pulse のデジタル感がさらに断片的 |
| **Bass Vocoder** | Saw | 低域バンドゲイン強調・LP 3kHz。ベースラインに声を乗せる |
| **Sci-Fi Voice** | *カスタム（半波整流 sine）* | Frmt +5・Q=6。非対称波形によるエイリアン的質感 |
| **Polar Glow** | Triangle | 上中域を緩やかに持ち上げたバンドゲイン・極スロー LFO（Band Tilt）。柔らかく輝くような質感 |
| **Megaphone** | Pulse | HP 400Hz / LP 4.5kHz（電話帯域）・中域バンドゲイン強調。Pulse の鼻声感でメガフォンらしさが増す |
| **Crystal** | *カスタム（sine + 3・5倍音）* | 高域バンドゲイン強調・Frmt +4。ベル的な倍音構造で輝くような高音 |
| **Monster** | Square | Frmt -6・低域強調。Square の厚みがオクターブ -2 の重さを増幅 |
